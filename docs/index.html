What this project is. A 6-piston pneumatic test rig driven by a TwinCAT PLC and controlled from a TcHmi web HMI. Each piston is a SOL-Spring two-position actuator — a single Extend coil, mechanical spring returns it — driven by one instance of FB_SolSpringPiston_2Pos. FB_MasterAutoCycle coordinates up to three pairs of (Sep, Push) pistons in parallel with the robot's Pos1/Pos2 signals. The rig exchanges "in position 1/2/3" plus AUTO_STARTED / PUSH_DONE / PISTONS_ERROR / RESET_ERROR frames with the robot over TCP/IP — the robot is the TCP server, the socket layer lives in a Python bridge (RobotBridge/robot_bridge.py) that dials in and translates each direction into ADS reads/writes, deliberately avoiding Beckhoff's paid TF6310 TCP/IP supplement.

How to open these docs:
Fastest (daily): double-click scripts\serve-docs.bat (or run scripts\serve-docs.ps1). It launches python -m http.server 8765 from the repo root and opens your default browser at http://127.0.0.1:8765/docs/index.html. Leave the terminal window open while browsing; Ctrl+C to stop.
Manual: cd C:\Users\Natan.Moshkin\source\repos\Flower && python -m http.server 8765, then browse to http://127.0.0.1:8765/docs/index.html.
Direct file:// open: works for the tabbed HTML pages, but the .md viewer needs HTTP (browsers block fetch() under file://) — the viewer will show a helpful error with the fix.
